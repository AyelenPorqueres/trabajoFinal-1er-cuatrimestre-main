¿QUE ES BE ORGANIZED?

Be organized es una aplicación web diseñada para mejorar tu organización personal, familiar y/o laboral. En la misma podras crear tareas, dividirla en categorias, establecer recordatorios e incluso tendras un calendario para visualizarlas de manera más rápida y sencilla. Incluye una opción para poder compartila con tu familia, amigos o compañeros de trabajo a traves de e-mail, al cual le llegaran las notificaciones de las tareas a realizar.
